CLASS 2: FILE/DIRECTORYの操作

# file/directory
file・directoryは階層構造で配置されている 特定のfile・directoryの位置は絶対パス又は相対パスで指定する 相対パスではcurrent directory (=現在の作業位置)を起点とする login直後のcurrent directoryはuserごとのhome directoryである

pwdコマンド

current directoryの確認

cdコマンド

cd DIR DIRに移動する cd home directoryに移動する cd - 直前のcurrent directoryに移動する

lsコマンド

ls current directory内のfile・directoryのlistを表示する ls DIR DIR内のfile・directoryのlistを表示する ls -l 詳細情報表示 ls -a ドットから始まるfile・directory名のものも表示する

catコマンド

cat FILE FILEの内容を表示する

lessコマンド

less FILE FILEの内容を表示する (１画面ずつ), qで終了

cpコマンド

cp FILE1 FILE2 FILE1をFILE2という名前でコピー cp -r DIR1 DIR2 directoryのコピー

mvコマンド

mv FILE DIR FILEをDIR内に移動 mv FILE1 FILE2 ファイル名をFILE1からFILE2に変更する

rmコマンド

rm FILE FILEを削除する rm -r DIR DIRを削除する

mkdirコマンド

mkdir DIR DIRを作製する
